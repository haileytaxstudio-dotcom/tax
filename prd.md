# 학습지 자동 발송 SaaS PRD

## 1. 개요 (Overview)

### 1.1 제품 목적

본 SaaS는 **학습지를 자동으로 발송**하고, **학습자의 제출 및 학습률을 어드민에서 관리·분석**할 수 있도록 돕는 서비스이다. 운영자는 학습 콘텐츠와 발송 주기를 설정하기만 하면, 학습 시작일부터 자동으로 학습 자료가 발송되고 학습 현황이 추적된다.

### 1.2 핵심 가치

* 운영자의 반복 업무 최소화 (발송, 알림 자동화)
* 학습자의 이탈 방지 (미제출 알림)
* 학습 현황의 시각적·정량적 관리
* 커뮤니티 기반 학습 동기 부여 (타인의 진행률/답안 열람)

---

## 2. 타겟 사용자 (Target Users)

### 2.1 관리자 (Admin)

* 학습지/시험지를 판매하거나 운영하는 개인 또는 교육 사업자
* 학습 일정, 발송 주기, 학습 현황을 한 곳에서 관리하려는 사용자

### 2.2 학습자 (Student)

* 카카오톡을 통해 학습지를 수령하고 제출하는 사용자
* 다른 학습자의 진행 상황을 참고하며 학습 동기를 얻고 싶은 사용자

---

## 3. 주요 기능 (Core Features)

## 3.1 카카오톡 비즈니스 채널 연동

### 설명

* 카카오톡을 주 발송 채널로 사용
* 문자/카카오 알림톡 대행 사이트의 API를 이용 (초기 버전에서는 구조만 고려)

### 기능 범위

* 학습 시작 알림 발송
* 학습지 발송
* 미제출 알림 발송

> ⚠️ 실제 API 연동은 추후 진행하며, 현재는 **연동을 고려한 구조 설계만 진행**

---

## 3.2 학습지 자동 발송 로직

### 기본 플로우

1. 관리자가 학습자의 정보를 어드민에서 입력
2. 학습 시작일 도래
3. 카카오톡으로 학습 시작 알림 발송
4. 설정된 주기에 따라 학습지 자동 발송

### 조건부 발송

* **시험지 미제출 시**: 알림 발송 (카카오톡)
* **시험지 제출 시**: 다음 학습일에 다음 학습 자료 발송

---

## 3.3 시험지 제출 및 관리

### 학습자 기능

* 학습지 확인
* 답안지 파일 업로드 (이미지/PDF 등)
* 제출 상태 확인

### 상태 값

* 미제출
* 제출 완료
* 확인 완료 (관리자 확인용, 선택사항)

---

## 3.4 결제 및 학습 시작 관리

### 결제 방식

* 결제는 **외부에서 직접 처리** (PG 연동 없음)

### 관리자 입력 정보

* 이름
* 연락처 (카카오톡 발송 기준)
* 학습 시작일
* 학습 과정(커리큘럼)

> 관리자가 직접 입력 시 해당 학습자는 즉시 자동화 플로우에 편입됨

---

## 3.5 학습 상황 분석 페이지 (학생용)

### 주요 기능

* 전체 학습자 진행률 확인 (퍼센트 기반)
* 자신의 진행률 표시
* 다른 학습자의 평균 진행률 확인

### 커뮤니티 요소

* 다른 학습자가 업로드한 답안지 열람 가능
* 익명 또는 닉네임 기반 표시

> 목적: 비교를 통한 학습 동기 부여

---

## 3.6 어드민 페이지

### 3.6.1 대시보드

* 전체 학습자 수
* 평균 학습 진행률
* 미제출 인원 수

### 3.6.2 학습자 관리

* 학습자 리스트
* 학습 시작일 수정
* 학습 상태 확인

### 3.6.3 학습지 관리 탭 (핵심 기능)

#### 기능 설명

* 학습지/시험지 파일 업로드
* 발송 주기 설정

#### 설정 항목

* 학습지명
* 파일 업로드 (PDF, 이미지 등)
* 발송 기준일

  * 학습 시작일 기준 N일차
  * 이전 학습지 제출 완료 후 N일
* 미제출 알림 기준 시간

> 어떤 파일이 **언제**, **어떤 조건에서** 발송되는지 관리자가 직접 설정 (학습일 기준일로 며칠마다 주기를 설정하도록)

---

## 4. 유저 플로우 (User Flow)

### 관리자 플로우

1. 학습지 업로드 및 발송 주기 설정
2. 학습자 정보 입력 (회원가입 기능 미구현)
3. 학습 시작일 도래 → 자동 발송
4. 학습 현황 모니터링

### 학습자 플로우

1. 카카오톡으로 학습 시작 안내 수신
2. 학습지 수령
3. 답안지 제출
4. 다음 학습지 수령

## 7. 성공 지표 (Success Metrics)

* 학습자 평균 제출률
* 학습 완료율
* 미제출 알림 후 제출 전환율
* 관리자 운영 시간 감소율

---

## 8. MVP 범위 요약

* 관리자 수동 학습자 등록
* 학습지 자동 발송
* 제출 여부에 따른 알림
* 기본 학습 진행률 대시보드

> MVP 목표: **"학습지를 자동으로 보내고, 누가 공부하고 있는지 한눈에 보이게 하는 것"**
